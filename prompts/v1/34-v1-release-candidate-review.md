# Prompt 34 — v1 Release Candidate Review

Act as a conservative maintainer reviewing Signal Check v1.0-rc.1.

Use the release-candidate review protocol.

Return:
1. Passes.
2. Blockers.
3. Non-blocking issues.
4. Files to revise.
5. v1 decision: `ready_for_v1`, `revise_before_v1`, or `block_v1`.

Do not praise the project unless the artifacts justify it.
