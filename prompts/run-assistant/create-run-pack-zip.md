# Create Run-Pack ZIP

Use this prompt inside the assistant when Data Analysis is available.

## Prompt

Create a ZIP file containing the completed Signal Check run pack.

The ZIP must contain exactly these files:

- 01-product-brief.md
- 02-sampling-plan.yaml
- 03-stimuli.yaml
- 04-negative-controls.yaml
- 05-prompt-robustness-plan.yaml
- 06-raw-reactions.md
- 07-synthesis.md
- 08-bias-audit.yaml
- 09-claim-safety-review.md
- 10-smallest-real-anchor.md
- 11-run-review.yaml
- 12-run-index.md

Before creating the ZIP, verify:

- no demand validation is claimed
- no representative findings are claimed
- no synthetic testimonials are included
- negative controls are present
- kill criteria are present
- Smallest Real Anchor is present
