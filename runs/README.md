# Local Runs
This folder is reserved for local Signal Check run work.
Do not commit private or internal runs from this folder.
For public teaching examples, use:
```text
examples/<example-id>/
```

For private or internal runs, prefer a folder outside the repository, for example:

```text
../_local-runs/signal-check/<run-id>/
```

The repository should remain a public method kit: docs, templates, prompts, schemas, quality gates, and public-safe examples.
