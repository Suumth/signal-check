# Run Assistant Regression Checklist

## Setup

- [ ] Assistant states it is optional.
- [ ] Assistant states it is not a validation oracle.
- [ ] Assistant asks one focused question at a time.
- [ ] Assistant does not invent missing product facts.

## Test material

- [ ] Strong-fit perspective included.
- [ ] Adjacent-fit perspective included.
- [ ] Skeptical-fit perspective included.
- [ ] Poor-fit perspective included.
- [ ] Non-market-control perspective included.
- [ ] Negative controls included.
- [ ] Kill criteria included.

## Analysis

- [ ] Raw reactions are not called real user evidence.
- [ ] Control behavior is analyzed.
- [ ] Counter-evidence is included.
- [ ] Claim safety is reviewed.
- [ ] Bias audit is included.

## Output

- [ ] Smallest Real Anchor is concrete.
- [ ] Stop or narrow criteria are defined.
- [ ] No demand or purchase-intent proof is claimed.
- [ ] 12-file run pack is complete.
