# Custom GPT Upload Checklist

Use this before publishing or updating the optional Signal Check Run Assistant.

## Before upload

- [ ] Instructions are under the current builder limit.
- [ ] Instructions define the assistant as a facilitator, not a validation oracle.
- [ ] Knowledge files are the six upload-ready files.
- [ ] No private product data is included.
- [ ] No raw personal data is included.
- [ ] No app, website, or private note exports are included.

## After upload

- [ ] Ask the assistant to summarize its role.
- [ ] Ask the assistant what it must not claim.
- [ ] Ask the assistant to start a new run.
- [ ] Ask the assistant to stop on a negative-control failure.
- [ ] Ask the assistant to output a Smallest Real Anchor.
- [ ] Verify that it does not claim validation or representative findings.
