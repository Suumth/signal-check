# Claim Safety Final Checklist

Block release if public docs imply:

- [ ] Signal Check replaces users.
- [ ] Block if synthetic output proves demand.
- [ ] Results are representative.
- [ ] Block if purchase intent is validated.
- [ ] Block if synthetic testimonials are treated as acceptable.
- [ ] A run measures market size.
- [ ] A run predicts conversion.

Accept only if limitations are explicit and visible.
