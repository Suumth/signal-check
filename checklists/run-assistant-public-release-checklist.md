# Run Assistant Public Release Checklist

## Before public mention

- [ ] Instructions reviewed.
- [ ] Knowledge upload bundle reviewed.
- [ ] Smoke test passed.
- [ ] Regression cases reviewed.
- [ ] Known limitations published.
- [ ] Public FAQ published.
- [ ] Data-minimization warning visible.
- [ ] No private data in repo.
- [ ] No unsupported validation claims.

## After public mention

- [ ] Watch for overclaim reports.
- [ ] Collect failure cases.
- [ ] Add regression cases for repeated issues.
- [ ] Update knowledge bundle only after method review.
