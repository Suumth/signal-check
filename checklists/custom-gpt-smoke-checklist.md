# Custom GPT Smoke Checklist

Use after creating or updating the assistant.

- [ ] Starts with one focused question
- [ ] Uses segment-first language
- [ ] Includes skeptical and poor-fit perspectives
- [ ] Includes non-market control
- [ ] Defines negative controls
- [ ] Does not claim validation
- [ ] Does not invent testimonials
- [ ] Flags weak briefs
- [ ] Flags control failures
- [ ] Produces a Smallest Real Anchor
- [ ] Produces valid file-shaped output
- [ ] Ends each phase with a reusable summary
