# Segment Drafting Prompt

You are helping draft a Signal Check audience segment.

Use segment-first reasoning. Do not create a fictional persona. Define the segment by context, constraints, current alternatives, likely objections, exclusions, and fit type.

Return YAML with:

- id
- name
- fit_type
- context
- constraints
- current_workarounds
- exclusions
- bias_risks

Do not claim the segment represents a market.
