# v1 Known Limitations

Signal Check v1 is intentionally limited.

## Limitations

- It depends on careful human operation.
- It cannot prevent all misuse.
- It does not collect real user evidence.
- It does not quantify demand.
- It does not replace interviews, experiments, or pilots.
- It can still reflect prompt bias or operator bias.
- It works best when users welcome skeptical output.

## Responsible use

Use Signal Check before building to reduce self-convincing. Do not use it as proof that a product should be built.
