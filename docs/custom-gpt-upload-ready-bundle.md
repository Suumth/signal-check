# Custom GPT Upload Ready Bundle

This document defines the upload-ready knowledge set for the optional Signal Check Run Assistant.

The assistant is optional. Signal Check remains a file-based method kit that can be used manually.

## Upload exactly these knowledge files

Use the six files in:

```text
knowledge/signal-check-run-assistant/upload-ready/
```

Recommended upload order:

1. `01-rules.md`
2. `02-methodology-prompts.md`
3. `03-run-pack-templates-lite.md`
4. `04-segments-panel-general.md`
5. `05-phase-map.md`
6. `06-example-quiet-invoice.md`

## Do not upload

Do not upload:

- private project notes
- customer data
- internal research notes
- raw transcripts with personal data
- app source code
- website source code
- large unreviewed run outputs
- media assets

## Why this bundle exists

Custom GPT knowledge retrieval is more reliable when the files are:

- few in number
- self-contained
- consistently named
- short enough for targeted retrieval
- explicit about method boundaries

## Canonical sources

The canonical method remains the public repository. These upload-ready files are a convenience bundle for the optional assistant.
