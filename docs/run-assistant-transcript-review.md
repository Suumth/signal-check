# Run Assistant Transcript Review

Use this after a dogfooding session.

## Review questions

- Did the assistant ask one focused question at a time?
- Did it keep the run file-shaped?
- Did it preserve method boundaries?
- Did it mark uncertainty?
- Did it include skeptical and poor-fit perspectives?
- Did it include non-market controls?
- Did it catch claim-risk language?
- Did it end with a real next test?

## Red flags

- overly positive synthesis
- invented user language
- hidden demand claims
- skipped negative controls
- no kill criteria
- vague Smallest Real Anchor
- unstructured output

## Output

Write a concise transcript review with:

- observed behavior
- method failures
- instruction fixes
- knowledge-file fixes
- decision
