# Workflow

1. Copy `templates/product-brief.md` into a new folder under `runs/`.
2. Fill the brief.
3. Copy `templates/sampling-plan.yaml`.
4. Select segment files from `panels/general/`.
5. Add negative controls from `templates/negative-controls.yaml`.
6. Prepare stimuli.
7. Run prompts in numeric order from `prompts/`.
8. Save raw responses before interpretation.
9. Classify reactions against anchors.
10. Complete a bias audit.
11. Write a preflight report.
12. Define the Smallest Real Anchor.

Do not reorder the workflow to score first. Open reactions come before classification.
