# Run Assistant Failure Mode Catalog

## Response collapse

All perspectives sound similar or positive.

Required response:

- flag collapse
- do not synthesize as signal
- revise prompts or segments
- add skeptical and poor-fit pressure

## Negative-control failure

A control receives strong positive reaction.

Required response:

- mark the run as unstable
- do not continue to confident synthesis
- inspect stimulus ambiguity
- revise controls and repeat

## Claim-risk synthesis

The synthesis implies demand, validation, representative findings, or purchase intent.

Required response:

- rewrite the claim
- add boundary note
- keep evidence directional
- move decision to real-world anchor

## Insufficient brief

The idea lacks segment, problem, alternative, or claim boundaries.

Required response:

- ask focused setup questions
- delay panel generation
- do not fill gaps with invention

## Missing real anchor

The run ends with vague advice instead of a real action.

Required response:

- define who sees what
- define what signal matters
- define stop or narrow criteria
