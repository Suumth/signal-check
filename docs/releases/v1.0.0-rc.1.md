# Signal Check v1.0.0-rc.1

Signal Check v1.0.0-rc.1 is the first release candidate for the stable file-based method kit.

## What it is

Signal Check helps builders examine product ideas before building by making segments, objections, risks, negative controls, prompt robustness, claim boundaries, and the next real test explicit.

## What it is not

Signal Check does not validate demand, replace real user research, produce representative findings, simulate real users, generate synthetic testimonials, or prove purchase intent.

## Included

- v1 method contracts
- manual run workflow
- run-pack templates
- JSON schemas
- prompt library
- starter panels
- public-safe examples
- claim guardrails
- quality gates
- contributor and maintainer docs

## Release status

This is a release candidate. It is suitable for public review, but not yet the final v1.0.0 release.
