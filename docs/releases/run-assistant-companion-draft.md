# Signal Check Run Assistant Companion — Draft Release Notes

The Signal Check Run Assistant is an optional Custom GPT companion for guided file-based runs.

## Included

- Custom GPT setup documentation
- upload-ready knowledge bundle
- GPT builder configuration
- output contract for 12-file run packs
- smoke test and regression scenarios
- dogfooding and maintenance guides
- public FAQ and known limitations

## Boundary

The assistant does not change Signal Check's method boundaries. It does not validate demand, replace real user research, generate synthetic testimonials, prove purchase intent, or produce representative findings.

## Status

Draft companion-release materials. Not a core-method release.
