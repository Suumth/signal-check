# Run Assistant Known Limitations

The optional Signal Check Run Assistant has important limits.

## It depends on the conversation

If the user provides vague input, the assistant should not invent missing facts.

## Knowledge retrieval can be imperfect

Uploaded knowledge files may not always be retrieved at the right moment. The instructions must include the most important boundaries directly.

## Long runs can drift

Long conversations can lose earlier context. Phase summaries reduce this risk but do not remove it.

## It cannot create evidence

Synthetic perspective reactions are not real user evidence.

## It cannot validate market demand

The assistant cannot validate demand, prove purchase intent, or produce representative findings.

## It may need maintenance

If regression cases fail, update instructions or knowledge files and document the change.
