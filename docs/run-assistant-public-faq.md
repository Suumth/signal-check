# Run Assistant Public FAQ

## Is the Run Assistant required?

No. Signal Check works as a file-based method kit. The assistant is optional.

## Does the assistant validate demand?

No. It helps structure preflight thinking and artifacts. It does not validate demand, prove purchase intent, or produce representative findings.

## Does it replace real user research?

No. It helps prepare better real-world tests. Real user research and real behavior remain separate.

## What does it produce?

It helps produce a run pack:

- product brief
- sampling plan
- stimuli
- negative controls
- prompt robustness plan
- raw synthetic reactions
- synthesis
- bias audit
- claim-safety review
- Smallest Real Anchor
- run review
- run index

## What should I not upload?

Do not upload private customer data, confidential product plans, personal raw transcripts, or internal source material unless you intentionally want that data processed.

## What is the best first use?

Use it on a small, non-sensitive product idea and check whether the assistant preserves method boundaries.
