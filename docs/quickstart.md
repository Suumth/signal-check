# Quickstart

1. Read `README.md`.
2. Copy `templates/product-brief.md` into `runs/my-idea/brief.md`.
3. Copy `templates/sampling-plan.yaml` into the same folder.
4. Choose at least four segment files from `panels/general/`.
5. Copy `templates/negative-controls.yaml`.
6. Write one or two stimuli.
7. Run the prompts in `prompts/` manually.
8. Save raw responses.
9. Complete `templates/preflight-report.md`.
10. Fill `templates/smallest-real-anchor.md`.

Use `examples/searcue/` as a shape reference.
