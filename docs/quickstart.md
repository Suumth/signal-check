# Quickstart

1. Read `README.md`.
2. Copy `templates/product-brief.md` into `runs/my-idea/brief.md`.
3. Copy `templates/sampling-plan.yaml` into the same folder.
4. Choose at least four segment files from `panels/general/`: fitting, adjacent, skeptical, and poor-fit.
5. Copy `templates/negative-controls.yaml`.
6. Add or identify a non-market control when false support is a risk.
7. Copy `templates/prompt-robustness-plan.yaml`.
8. Write one or two stimuli with explicit claim boundaries.
9. Run the prompts in `prompts/` manually.
10. Save raw responses before summary.
11. Classify reactions against anchors with evidence, counter-evidence, and uncertainty.
12. Complete a bias audit and apply kill criteria.
13. Complete `templates/preflight-report.md`.
14. Fill `templates/smallest-real-anchor.md`.

Use `examples/searcue/` as a shape reference.

v0.1 has no CLI, API, Web UI, engine, or model integration. Use the files manually.
