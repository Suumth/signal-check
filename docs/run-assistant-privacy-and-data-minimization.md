# Run Assistant Privacy and Data Minimization

The optional Run Assistant should be used with data-minimized inputs.

## Recommended inputs

- public or sanitized product idea
- short problem description
- non-sensitive segment assumptions
- public-safe example artifacts

## Avoid uploading

- private customer records
- raw personal transcripts
- confidential product strategy
- internal source code
- non-public commercial details
- personally identifiable information

## Repository boundary

The public repository must not include private runs, customer data, or internal research notes.

## Assistant boundary

The assistant can facilitate a run, but it should not become a storage location for sensitive research data.
