# Run Assistant Public Release Plan

The Signal Check Run Assistant is an optional companion for guided file-based runs.

It should be released only after:

- knowledge upload bundle is reviewed
- Custom GPT instructions are reviewed
- smoke tests pass
- regression scenarios pass or have documented repairs
- data-minimization warnings are visible
- public FAQ is ready
- known limitations are published

## Release channels

Recommended:

- repository documentation
- GitHub release note
- short public announcement
- no claims of validation or research automation

## Not recommended

- positioning as an AI market research tool
- publishing synthetic testimonials
- implying the assistant replaces customer discovery
- promising accuracy, coverage, or predictive power

## Release decision

Use:

- ready
- polish recommended
- blocked
