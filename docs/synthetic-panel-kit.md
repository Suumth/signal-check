# Synthetic Panel Kit

## Purpose

The Synthetic Panel Kit is the method layer that explains how Signal Check uses synthetic perspectives without pretending to perform real market research.

It helps a product builder examine an idea through structured perspectives before building. The output is not validation. The output is a clearer hypothesis, sharper objections, and a real next test.

## Core rule

A synthetic panel is a thinking scaffold, not an evidence source.

It can help ask better questions. It cannot prove demand, represent a market, simulate real customers, or replace user research.

## Required components

Every panel-kit run must include:

1. A product brief
2. A segment model
3. A panel composition
4. At least one skeptical perspective
5. At least one poor-fit or non-market-control perspective
6. Negative controls
7. Prompt-robustness checks
8. Bias audit
9. Synthesis with counter-evidence
10. Smallest Real Anchor

## Panel lifecycle

```text
idea → product brief → segment model → panel composition → controlled reactions → bias audit → synthesis → real-test translation
```

## Non-goals

Signal Check does not produce:

- Representative findings
- Forecasts
- Purchase-intent validation
- No real customer quotes
- No synthetic testimonials
- Statistical confidence
- Product-market fit scores

## Output standard

A good output should make the next real-world test smaller, clearer, and more honest.
